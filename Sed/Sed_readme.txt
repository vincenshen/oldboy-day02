## 功能特性
* 可以对全文查找指定的字符串
* 可以对全文替换指定的字符串
* 支持re正则表达式

## 使用方法及注意事项
### 运行环境
* 该程序使用python3版本编写，不兼容python2版本

### 使用方法
* python sed.py -h     打印帮助信息
* python sed.py -f {str} {file_name}     全文查找字符
* python sed.py -r {old_str} {new_str} {file_name}   全文替换字符串
